Technical Education & Training

Computer Science and Web Technologies

Self-directed, open-source technical education
Modeled after a Bachelor of Science curriculum

This page documents my self-directed technical education and training in Computer Science and Web Technologies. The program is structured and modeled after a traditional Bachelor of Science curriculum, with emphasis on systems, programming, web infrastructure, and applied cybersecurity.

This education is non-accredited and independently pursued. It is built using an open-source learning model inspired by initiatives such as the Open Source Society University (OSSU), combining structured coursework, documented reading, hands-on labs, and real-world projects.

The curriculum integrates computer science fundamentals with modern web technologies, including Linux systems, networking concepts, scripting, web architecture, performance, and security-informed engineering practices.

As of January 2026, this program is approximately 90% complete. Remaining components include final documentation, expanded lab write-ups, and continued applied security projects.

Transparency is a core principle of this work. Rather than presenting credentials without evidence, this site exists to make learning visible — including curriculum structure, progress tracking, and supporting materials.
